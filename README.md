# 🌐 FullStack Application - Spring Boot + Vanilla JavaScript

A **full-stack web application** with:
- **Backend:** Spring Boot REST API
- **Frontend:** Static HTML, CSS, and Vanilla JavaScript (fetch API)
- **Database:** ArrayList Storage ( not using Database)

This project demonstrates a complete CRUD workflow, REST API integration, authentication, and responsive UI without any frontend frameworks.

---

## 📌 Features
- **Backend (Spring Boot)**
  - RESTful API with Spring Web
  - Validation and exception handling
- **Frontend (Vanilla JS)**
  - Static HTML/CSS pages
  - Fetch API for AJAX calls
  - DOM manipulation for dynamic content
  - Responsive design with CSS Flex/Grid
  

---

## 🛠 Tech Stack
**Backend:** Spring Boot, Spring Data JPA, Spring Security, Maven  
**Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)  
**Database:** MySQL / PostgreSQL  
**DevOps:** Docker, Docker Compose  
**Testing:** JUnit, Mockito
